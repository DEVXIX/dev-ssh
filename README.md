# SSH/FTP Manager

idk what to call this but its a web-based server manager with ssh and file manager, database view/manage capabilities similar to termix. but it's a very old project i've worked on, the main purpose of it is to be self hosted.



## Installation

```bash
bun i
```

## Usage

```bash
bun run dev
```

and you're ready to go!

## Database

This application uses SQLite for local data storage
